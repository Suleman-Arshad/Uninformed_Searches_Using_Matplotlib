Grid Pathfinding Visualizer (Uninformed Search Edition)This repository contains a collection of Python scripts that visualize classic Uninformed Search algorithms on an 8x8 grid. Each script demonstrates how different data structures (Queues, Stacks, Priority Queues) and constraints affect pathfinding behavior and efficiency.
